# Contributing — AYA-BLOX-6551

This is a Level 1 reference implementation. Its purpose is educational and documentary — a minimal, forkable proof of the ERC-6551 game mechanic.

## What this repo is for

A fixed-scope reference implementation of the ERC-6551 game mechanic. Documentation corrections and bug reports are welcome — contract logic and feature additions are not.

## Found a bug?

Open a GitHub issue with:
- The exact function and line number
- A minimal reproduction (forge test case preferred)
- Expected vs. actual behaviour

## Want to extend the mechanic?

Fork the repo — that is what it is here for. If you build something on top of it, a mention or link back is appreciated but not required.

## Pull requests

PRs that change contract logic will not be merged into this repo. This implementation tracks a specific experiment with a fixed scope. Open an issue first if you want to discuss a change — if it belongs in the reference, it will be incorporated in a versioned update.

PRs for documentation corrections, broken links, or README improvements are welcome.

## Security issues

Do not open a public issue for vulnerabilities. See [SECURITY.md](./SECURITY.md) for the reporting process.
